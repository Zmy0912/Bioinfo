# Pi 滑动窗口基因标注程序 - 图形化界面版本

## 版本信息

**版本**: v2.0 (GUI)
**发布日期**: 2026-03-07

## 新功能

本图形化界面版本在原有命令行版本的基础上，新增了以下功能：

### 1. 用户友好的图形化界面
- 基于 tkinter 构建的现代化界面
- 直观的参数设置界面
- 实时运行日志显示
- 进度指示器

### 2. 多种输出格式支持
- **PNG 格式**: 位图格式，适合屏幕查看
- **SVG 格式**: 矢量图格式，无损缩放，适合打印和出版
- **PDF 格式**: 矢量格式，适合文档嵌入和打印
- **全部格式**: 同时生成 PNG、SVG 和 PDF 三种格式

### 3. 所有原有功能
- 解析 Pi 滑动窗口数据文件
- 解析 GFF3 基因注释文件
- 三种显著性判断标准（相对、绝对、下限）
- 智能标签布局
- 多条件基因优先级排序
- 生成详细的分析报告

## 使用方法

### 启动程序

```bash
python pi_plot_gui.py
```

### 界面操作流程

1. **选择文件**
   - 点击"浏览..."按钮选择 Pi 文件
   - 点击"浏览..."按钮选择 GFF3 文件
   - 点击"浏览..."按钮选择输出目录

2. **设置输出格式**
   - 选择需要的输出格式（PNG/SVG/PDF/全部）
   - 建议学术出版使用 SVG 或 PDF 格式

3. **调整绘图参数**
   - 输出文件名：默认为 `pi_sliding_window_plot`
   - 字体大小：8-18（推荐 10-12）
   - 最大基因数：5-50（推荐 10-15）
   - 窗口大小：1000-100000 bp（推荐 10000-20000）
   - 相对显著性：0.5-5.0（推荐 1.5-3.0）
   - 绝对阈值：可选，用于绝对显著性筛选
   - 下限阈值：可选，用于检测最低 Pi 值基因

4. **开始分析**
   - 点击"开始分析"按钮
   - 查看运行日志了解处理进度
   - 分析完成后会弹出提示框

5. **查看结果**
   - 在输出目录中查看生成的图表文件
   - 查看 `_info.txt` 文件获取详细分析结果

## 输出格式对比

### PNG 格式
- **优点**: 文件体积小，查看方便
- **缺点**: 缩放后失真
- **适用场景**: 屏幕查看、快速预览

### SVG 格式（矢量图）
- **优点**: 无损缩放，适合大尺寸打印
- **缺点**: 文件体积较大
- **适用场景**: 学术出版、海报、高质量打印

### PDF 格式
- **优点**: 矢量格式，适合文档嵌入
- **缺点**: 编辑需要专业软件
- **适用场景**: 论文文档、报告、打印

### 全部格式
- 同时生成三种格式
- 适合需要多种用途的场景

## 参数调优建议

### 窗口大小 (--window-size)
- **小窗口（5000-10000 bp）**：适合检测局部变化
- **中窗口（10000-20000 bp）**：平衡局部和全局特征（推荐）
- **大窗口（20000-50000 bp）**：检测大尺度的趋势

### 显著性标准 (--significance-std)
- **1.5**：宽松标准，标注更多基因
- **2.0**：标准平衡（默认）
- **2.5-3.0**：严格标准，仅标注最显著基因

### 输出格式选择建议

| 使用场景 | 推荐格式 | 原因 |
|---------|---------|------|
| 屏幕查看 | PNG | 加载快，文件小 |
| 学术论文 | SVG 或 PDF | 矢量格式，无损缩放 |
| 海报展示 | SVG | 矢量格式，打印质量高 |
| 文档嵌入 | PDF | 兼容性好 |
| 多用途 | 全部格式 | 满足各种需求 |

## 常见问题

### Q1: 程序启动失败，提示找不到 tkinter

**解决方案**:
- Windows: 重新安装 Python，确保勾选 "tcl/tk and IDLE" 选项
- Linux: `sudo apt-get install python3-tk`
- macOS: 使用 Homebrew 安装: `brew install python-tk`

### Q2: 生成的图表文字显示为方框

**解决方案**:
- 程序已内置中文字体支持，但可能需要安装相应字体
- Windows: 程序使用 SimHei 或 Microsoft YaHei
- Linux: 安装中文字体包 `sudo apt-get install fonts-wqy-zenhei`

### Q3: 选择 SVG 或 PDF 格式后，文件很大

**解决方案**:
- 减少标注的基因数量
- 减小字体大小
- 这是正常现象，矢量格式的特点

### Q4: 界面卡死，没有响应

**解决方案**:
- 程序在后台线程运行分析，界面应该不会卡死
- 如果遇到问题，请检查文件大小和数据格式
- 大型基因组可能需要较长处理时间

### Q5: 如何获得高质量的图表用于发表？

**解决方案**:
- 选择 SVG 或 PDF 格式
- 调整参数使图表清晰（减少基因数、适当字体大小）
- 使用矢量图编辑软件（如 Adobe Illustrator、Inkscape）进一步编辑
- 在 LaTeX 文档中，PDF 格式可以直接嵌入

## 技术细节

### 矢量图优势

SVG 和 PDF 格式是基于矢量的图形格式，具有以下优势：

1. **无损缩放**：可以放大任意倍数而不失真
2. **文件体积小**：相对于同等分辨率的位图
3. **可编辑性**：可以在矢量图编辑软件中修改
4. **打印质量高**：适合高质量打印
5. **适合出版**：学术期刊通常要求矢量格式

### 代码结构

- `PiPlotProcessor`: 数据处理和绘图核心类
  - `parse_pi_file`: 解析 Pi 文件
  - `parse_gff3_file`: 解析 GFF3 文件
  - `find_significant_genes`: 查找显著基因
  - `plot_pi_with_genes`: 绘制图表

- `PiPlotGUI`: 图形化界面类
  - `setup_ui`: 设置界面布局
  - `browse_*_file`: 文件浏览功能
  - `start_analysis`: 开始分析
  - `run_analysis`: 运行分析（后台线程）
