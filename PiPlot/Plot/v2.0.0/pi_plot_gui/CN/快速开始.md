# 快速开始指南

## 一、安装步骤

### 1. 检查 Python 环境

打开命令提示符（CMD）或 PowerShell，输入：

```bash
python --version
```

确保 Python 版本为 3.6 或更高。

### 2. 安装依赖

```bash
pip install pandas matplotlib numpy
```

或使用 requirements.txt：

```bash
pip install -r requirements.txt
```

### 3. 启动程序

**方法 1：双击启动**
- 双击 `启动程序.bat` 文件

**方法 2：命令行启动**
```bash
python pi_plot_gui.py
```

## 二、快速使用示例

### 示例 1：首次使用（推荐）

1. 启动程序
2. 选择 Pi 文件
3. 选择 GFF3 文件
4. 选择输出目录
5. 保持默认参数
6. 选择 "PNG格式"（快速预览）
7. 点击 "开始分析"

### 示例 2：学术出版用途

1. 启动程序
2. 选择文件和输出目录
3. 选择 "全部格式"（PNG+SVG+PDF）
4. 设置字体大小为 10
5. 设置最大基因数为 12
6. 点击 "开始分析"
7. 在输出目录中使用 SVG 或 PDF 格式用于发表

### 示例 3：检测最低 Pi 值基因

1. 启动程序
2. 选择文件和输出目录
3. 设置下限阈值为 0.01（根据实际情况调整）
4. 选择 SVG 格式
5. 点击 "开始分析"

## 三、输出文件说明

运行完成后，输出目录中会生成以下文件：

```
输出目录/
├── pi_sliding_window_plot.png    # PNG 格式图表（如果选择 PNG）
├── pi_sliding_window_plot.svg    # SVG 格式图表（如果选择 SVG）
├── pi_sliding_window_plot.pdf    # PDF 格式图表（如果选择 PDF）
└── pi_sliding_window_plot_info.txt  # 详细分析结果
```

### 图表文件说明

| 格式 | 文件扩展名 | 用途 | 编辑软件 |
|------|----------|------|---------|
| PNG | .png | 屏幕查看 | 任何图片查看器 |
| SVG | .svg | 学术出版、打印 | Inkscape、Adobe Illustrator |
| PDF | .pdf | 文档嵌入 | Adobe Acrobat、Inkscape |

### 信息文件说明

`_info.txt` 文件包含：
- 分析参数设置
- 找到的显著基因总数
- 每个显著基因的详细信息：
  - 基因名称
  - 基因位置
  - Pi 值
  - Z-score
  - 显著性类型

## 五、故障排除

### 问题 1：程序无法启动

**解决方案**：
1. 确认 Python 已安装：`python --version`
2. 确认依赖已安装：`pip list | findstr pandas matplotlib numpy`
3. 检查 tkinter 是否可用：`python -c "import tkinter"`

### 问题 2：中文显示为方框

**解决方案**：
- Windows: 已内置支持，应该正常显示
- Linux: 安装中文字体 `sudo apt-get install fonts-wqy-zenhei`
- macOS: 已内置支持，应该正常显示

### 问题 3：输出文件非常大

**解决方案**：
- 减少标注的基因数量（设置 max_genes 为 10-15）
- 减小字体大小（设置 gene_fontsize 为 8-10）
- SVG/PDF 格式文件较大是正常现象

### 问题 4：分析速度慢

**解决方案**：
- 增大窗口大小（设置 window_size 为 15000-20000）
- 减少标注的基因数量
- 关闭其他占用 CPU 的程序

## 八、快捷操作技巧

1. **批量处理**：如果需要处理多个文件，可以使用命令行版本（v1.1.0）编写批处理脚本

2. **保存参数**：记录成功的参数配置，以便复现结果

3. **预览调整**：先用 PNG 格式快速预览，确认效果后再生成 SVG/PDF

4. **多格式输出**：选择 "全部格式"，一次性生成三种格式，满足各种需求

5. **检查结果**：先查看 `_info.txt` 文件了解分析结果，再查看图表
