# 密码子使用频率分析工具 - 依赖项说明

## 项目简介

这是一个用于可视化密码子使用频率的Python工具，可以生成堆积柱状图，展示不同氨基酸对应密码子的RSCU值分布。支持多种配色方案、自定义文本格式、图表尺寸调整等高级功能，并提供独立显示的缩放和平移功能。

## 主要功能

- **数据导入**: 支持 .xlsx 和 .xls 格式的Excel文件
- **可视化**: 生成交互式堆积柱状图展示密码子RSCU值
- **多种配色**: 提供11种配色方案（tab20c, tab20b, tab20, Set1, Set2, Set3, Pastel1, Pastel2, viridis, plasma, inferno）
- **文本自定义**: 可调整柱状图内密码子名称、坐标轴标签、刻度标签、图例文字的字号
- **图表自定义**: 可调整画布大小、堆叠块高度/宽度、边框粗细、堆叠块距离X轴的距离
- **交互操作**: 支持缩放、平移等操作
- **多格式导出**: 支持PDF、SVG、PNG三种格式导出

**错误信息**: `ModuleNotFoundError: No module named 'openpyxl'`

**解决方案**:
```bash

```

### 3. 中文字体显示问题

**错误信息**: 图表中中文显示为方框或乱码

**解决方案**:
- **Windows**: 确保系统已安装 SimHei 或 Microsoft YaHei 字体
- **macOS**: 安装中文字体包

**错误信息**: `FileNotFoundError` 或 `PermissionError`

**解决方案**:
- 确认文件路径正确
- 确保文件未被其他程序占用
- 尝试将文件另存为 .xlsx 格式

## 数据格式要求

Excel文件应包含以下列（支持英文或中文列名）：

| 英文列名 | 中文列名 | 说明 |
|---------|---------|------|
| Codon | 密码子 | 密码子序列（如 UUU、AUG 等）|
| Frequency | 频数/数量 | 密码子出现频数 |
| RSCU | RSCU | 相对同义密码子使用频率 |
| AminoAcid | 氨基酸 | 对应的氨基酸（可选，如缺失会自动推导）|

程序会自动识别列名，如果列名格式不同，请确保数据列按上述顺序排列。

**支持的密码子格式**: 程序使用RNA碱基（U），也兼容DNA碱基（T），会自动转换。

**氨基酸排序**: 程序按照标准顺序排列20种氨基酸：Phe, Leu, Ile, Met, Val, Ser, Pro, Thr, Ala, Tyr, His, Gln, Asn, Lys, Asp, Glu, Cys, Trp, Arg, Gly

| Codon | Frequency | RSCU |
|-------|-----------|------|
| TTT   | 150       | 1.20 |
| TTC   | 100       | 0.80 |
| TTA   | 80        | 0.80 |

| ...   | ...       | ...  |

**注意**: 程序会自动将DNA碱基（T）转换为RNA碱基（U）进行处理。

### 数据说明

- **Frequency（频数）**: 该密码子在序列中出现的实际次数
- **RSCU（相对同义密码子使用频率）**: 衡量密码子使用偏好性的指标
  - RSCU = 1.0：无偏好性（均等使用）
  - RSCU > 1.0：偏好使用该密码子
  - RSCU < 1.0：不偏好使用该密码子
- 每个氨基酸通常对应2-6个密码子（除甲硫氨酸Met和色氨酸Trp外）

## 界面功能说明

### 工具栏功能

**第一行 - 文件操作**
- 选择Excel文件：导入数据文件
- 保存为PDF：导出为矢量PDF格式
- 保存为SVG矢量图：导出为可编辑SVG格式
- 保存为PNG：导出为高分辨率PNG图片

**第二行 - 图表设置**
- 配色方案：选择11种配色方案
- 显示密码子名称：控制柱状图内是否显示密码子标签
- 显示堆叠块：控制底部是否显示密码子堆叠块图例
- 画布大小：调整图表的宽度和高度（默认18x13）
- 堆叠块高度：调整底部堆叠块的高度（默认0.12，范围0.05-0.3）
- 堆叠块宽度：调整底部堆叠块的宽度（默认0.6，范围0.1-1.0）
- 重新绘制：应用所有参数设置并重绘图表

**第三行 - 文本和边框设置**
- 柱状图文字字号：调整柱状图内密码子名称大小（默认10，范围6-24）
- 坐标轴字号：分别调整X轴和Y轴标签大小（默认14，范围8-30）
- 刻度字号：分别调整X轴和Y轴刻度标签大小（默认12，范围6-20）
- 边框粗细：调整图表边框和刻度线粗细（默认1.0，范围0.5-5.0）
- 堆叠块位置：调整底部堆叠块距离X轴的距离（默认0.22，范围0.05-0.5）
- 堆叠块文字字号：调整底部堆叠块内文字大小（默认8，范围4-16）

**绘图区域**
- 显示生成的密码子使用频率图表
- 支持鼠标滚轮缩放和拖拽平移
- 通过滚动条在大型图表中导航

### 导航工具栏功能

绘图区域上方提供了导航工具栏，包含以下功能：
- Home：返回原始视图
- Back/Forward：前进/后退视图历史
- Pan：平移工具（点击后拖拽图表）
- Zoom：缩放工具（框选区域进行放大）
- Save：保存当前视图

## 常见问题

### Q: 图表生成后没有显示？
A: 确保Excel文件包含正确的列名（Codon、RSCU等），可以运行 `--debug` 模式查看文件内容。

### Q: 如何修改图表的配色方案？
A: 在图形界面第二行的"配色方案"下拉框中选择不同的配色方案，然后点击"重新绘制"。

### Q: 导出的图表质量如何调整？
A: 程序默认使用300 DPI导出，如果需要更高或更低的分辨率，可以在代码中修改 `dpi` 参数（位于 save_as_pdf、save_as_png 等方法中）。

### Q: 如何调整密码子标签的显示？
A: 在图形界面第二行勾选/取消"显示密码子名称"复选框，或在第三行调整"柱状图文字字号"。

### Q: 支持哪些密码子编码格式？
A: 程序支持RNA编码（U）和DNA编码（T），例如 UUU 和 TTT 都会被识别为苯丙氨酸密码子。

### Q: 为什么图表没有随窗口大小改变？
A: 这是设计的特性。图表以固定大小显示，支持滚动和缩放，这样可以更好地查看图表细节。您可以使用导航工具栏的缩放功能或滚动条来导航大型图表。

### Q: 如何在Linux系统上安装tkinter？
A: 使用包管理器安装，例如：
- Ubuntu/Debian: `sudo apt-get install python3-tk`
- Fedora: `sudo dnf install python3-tkinter`
- Arch: `sudo pacman -S tk`
