# 快速开始指南

## 5分钟快速上手

### 第一步：启动程序

双击 `bed_converter.py` 文件，或在命令行输入：

```bash
python bed_converter.py
```

### 第二步：选择文件类型

- **BED 格式**: 如果您的文件格式为 `gene_id  start  end  type  phase`，选择 "BED格式"
- **GFF3 格式**: 如果您的文件是标准 GFF3 格式，选择 "GFF3格式"

### 第三步：打开文件

1. 点击 "浏览" 按钮
2. 选择您的文件
3. 点击 "加载文件" 查看内容

### 第四步：转换坐标

1. 点击 "转换" 按钮
2. 等待处理完成
3. 查看预览区域的结果

### 第五步：保存结果

1. 确认预览结果正确
2. 点击 "保存文件"
3. 选择保存位置和文件名
4. 完成转换！

## 测试示例

### GFF3 示例测试

## 关键提示

- ✅ 转换前先点击 "加载文件" 预览
- ✅ 转换后查看预览确认正确再保存
- ✅ 原始文件不会被修改
- ✅ 注释行会自动保留
- ✅ 特征间相对距离保持不变

## 格式速查

### BED 格式
```
基因ID  起始位置  结束位置  特征类型  [相位]
```

### GFF3 格式
```
染色体  来源  类型  起始  结束  得分  链  相位  属性
```
属性中需包含 `gene_id=xxx`
