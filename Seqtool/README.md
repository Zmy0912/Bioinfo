# Bioinformatics Sequence Tools

A collection of command-line tools for biological sequence data processing and analysis.

## Usage

All scripts can be executed directly. Simply open them and follow the on-screen prompts to operate.

## Available Tools

### text_to_uppercase
Convert all lowercase letters in sequence files to uppercase.

**Usage:** Run the script, input the file path when prompted.

---

### merge_sequences
Merge multiple sequences from a folder into a single file.

**Usage:** Run the script, specify the folder containing sequence files.

---

### extract_sequences
Extract one or more specified sequences from a multi-sequence file.

**Usage:** Run the script, provide the input file and the sequence IDs to extract.

---

### remove_sequences
Delete one or more specified sequences from a sequence file.

**Usage:** Run the script, provide the input file and the sequence IDs to remove.

---

### rename_sequences
Rename all sequences in a file.

**Usage:** Run the script, specify the input file and follow the renaming prompts.

---

### complement_sequences
Complement one or more sequences within a file, with options for complement only or reverse complement.

**Features:**
- Complement only
- Reverse complement

**Usage:** Run the script, provide the input file and select the desired operation.

---

### extract_sequence_info
Extract sequence IDs and descriptions from one or more sequences in a file.

**Usage:** Run the script, provide the input file to extract sequence information.

---

### fasta_cleaner_gui
A visual sequence processing tool for removing illegal characters from sequences.

**Usage:** Run the script and operate step by step through the visual interface.

---

## Quick Start

1. Open any of the scripts listed above
2. Follow the interactive prompts
3. Provide the required file paths and parameters
4. View the processed results

## Notes

- All tools are designed for interactive command-line usage
- Ensure input files are in supported sequence formats (FASTA, etc.)
- Back up important files before processing
